The most basic

<div class=”tilt”>

For the most basic tilt, you just have to add the “tilt” class to the element on which you want to apply the effect.

No need to specify strength separately; the default is '5'.

But you can still specify the strength using “tilt_strength”.

<div tilt_strength=”5”>
