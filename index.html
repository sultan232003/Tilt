The most basic

<< /span>div class=”tilt”>

To apply a basic tilt, just add the “tilt” class to the element.

No need to specify strength separately; the default is '5'.

But you can still specify the strength using “tilt_strength”.

<< /span>div tilt_strength=”5”>

Disable X

<< /span>div class=”tilt” tilt_x=”false”>

To apply the effect, simply add the “tilt” class to the desired element and set tilt_x="false".

No need to specify strength separately; the default is '5'.

But you can still specify the strength using “tilt_strength”.

<< /span>div tilt_strength=”5”>

Disable y

<< /span>div class=”tilt” tilt_y=”false”>

To apply the effect, simply add the “tilt” class to the desired element and set tilt_y="false".

<< /span>div tilt_strength=”5”>

Scale

<< /span>div class=”tilt” tilt_scale=”1.1”>

Just add the “tilt” class and set tilt_scale="1.1", or whatever scale size you like, but max is "2" 😅.

No need to specify strength separately; the default is '5'.

But you can still specify the strength using “tilt_strength”.

<< /span>div tilt_strength=”5”>

Start Y

<< /span>div class=”tilt” tilt_StartY=”40”>

Just add the “tilt” class and set tilt_StartY="40", or whatever start angle you like, other than "90" 😋.

No need to specify strength separately; the default is '5'.

But you can still specify the strength using “tilt_strength”.

<< /span>div tilt_strength=”5”>

Infinity

<< /span>div class=”tilt” tilt_infinity=”5 15 0.5”>

Just add the “tilt” class and set tilt_infinity="5 15 0.5", for layer numbers, strength, and smallest layer size.
